## 09-02-2022 Quiz 03

# Header Comment: Create a program that translate the proteins in the DNA chain.

------------------------------------------------------------------------

```.py
in_protein = input("Input Protein: ")

if in_protein=="A":
    print("T")
elif in_protein=="T":
    print("A")
elif in_protein=="C":
    print("G")
elif in_protein=="G":
    print("C")
else:
    print("Error")
    exit()
```
